# MacDevTools 功能说明

## 🎉 新增功能

### 🌏 中文本地化
- **完整的中文界面**: 所有菜单、按钮、提示信息均已翻译为中文
- **语言切换**: 在设置中可选择中文或英文界面
- **智能回退**: 如果翻译缺失，自动回退到英文

### ⌨️ 可配置快捷键
- **快捷键列表**: 设置中显示所有可用的快捷键及其功能说明
- **自定义编辑**: 点击编辑按钮可自定义任意快捷键
- **实时生效**: 修改后立即生效，无需重启应用
- **重置功能**: 一键重置所有快捷键为默认设置

### 📋 完整快捷键列表

#### 文件操作
- **⌘N**: 新建文件
- **⌘⇧N**: 新建文件夹  
- **⌘S**: 保存当前文件
- **⌘⇧S**: 保存所有文件
- **⌘W**: 关闭当前标签页

#### 导航操作
- **⌘1**: 切换到 JSON 工具
- **⌘2**: 切换到 XML 工具
- **⌘3**: 切换到 Base64 工具
- **⌘4**: 切换到 HTTP 工具

#### 界面控制
- **⌘⇧F**: 全局搜索
- **⌘B**: 切换资源管理器
- **⌘G**: 切换预览面板
- **⌘,**: 打开设置

#### 搜索功能
- **⌘F**: 在当前文件中查找

### 🔧 设置功能增强

#### 外观设置
- **主题切换**: 深色/浅色模式
- **语言选择**: 中文/英文界面切换
- **自动保存**: 自动保存文件更改

#### 数据管理
- **备份导出**: 导出所有文件、设置和快捷键配置
- **备份导入**: 恢复之前的完整备份
- **智能合并**: 导入时智能合并设置和快捷键

### 📝 备份格式
新的备份格式包含：
```json
{
  "fileSystem": { /* 所有文件数据 */ },
  "settings": {
    "theme": "dark|light",
    "language": "zh|en", 
    "autoSave": true|false
  },
  "shortcuts": { /* 所有快捷键配置 */ }
}
```

### 🎯 使用方法

1. **查看快捷键**: 打开设置 → 快捷键区域
2. **修改快捷键**: 点击编辑按钮，按下新的组合键
3. **保存配置**: 点击保存按钮或导出备份
4. **重置配置**: 点击"重置全部"恢复默认设置

### 🌟 优势

- **完全本地化**: 所有界面文本均支持中文
- **高度可定制**: 每个快捷键都可以自定义
- **用户友好**: 直观的界面和清晰的功能说明
- **数据安全**: 完整的备份和恢复机制
- **macOS 原生**: 保持 macOS 的设计规范和交互习惯
